Follow these steps if model import failed
1. Download the [project zip file](https://github.com/IBM/Developer-Playground/raw/agro-chatbot/Agro-Smart-Assistant/data/crop-recommendation.zip).
2. Login to your [IBM Cloud Pak for Data](https://dataplatform.cloud.ibm.com) account with the region given in your sandbox terminal. Click on "Create a Project".
  <img src = "https://raw.githubusercontent.com/IBM/Developer-Playground/master/didact/images/section_error_1.png" width = "750"/>
3. Click on "Create a project from sample or file".
  <img src = "https://raw.githubusercontent.com/IBM/Developer-Playground/master/didact/images/section_error_2.png" width = "750"/>
4. Upload the zip file that was just downloaded in Step 1 > Enter a project name > click "Create".
  <img src = "https://raw.githubusercontent.com/IBM/Developer-Playground/master/didact/images/section_error_3.png" width = "750"/>
5. After the project is created, click on "View new project".
  <img src = "https://raw.githubusercontent.com/IBM/Developer-Playground/master/didact/images/section_error_4.png" width = "750"/>
6. Click on the Assets tab.
  <img src = "https://raw.githubusercontent.com/IBM/Developer-Playground/master/didact/images/section_error_5.png" width = "750"/>
